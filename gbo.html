Green Bank Observatory

A quick look into my personal involvement with the Green Bank Observatory.

Detecting low-frequency Gravitational Waves using Pulsar Timing.

Define GWaves and pulsars and the data sets.

Talk about PSR. Talk about work done - mini LIGO, laser calibration. Data set analysis on HPCC, curve fitting, modelling and classification.
